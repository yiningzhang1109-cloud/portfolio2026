The Problem

AI music generation tools have a specific failure point. They perform well when given precise musical language: tempo, instrumentation, harmonic character, dynamic range. They perform poorly when given emotional descriptions of scenes — which is exactly how creators naturally think about the music they need.

The gap between how creators think about music and how these tools actually process input is where most attempts break down. This tool is designed to close that gap at the exact point where it opens.

01

Scene Analysis & Reference Discovery

The user describes a scene. The tool analyzes it across four dimensions: emotion, rhythm, tension, and theme. It then surfaces fourteen reference tracks from real film and game scores — seven that match the scene's emotional register, and seven that work in deliberate contrast against it. Each reference includes a direct link so the user can listen rather than imagine.

02

Parameter Extraction & Prompt Generation

The user selects the tracks that feel right. The tool extracts shared musical parameters across those choices: mode, tempo range, core instrumentation, rhythmic texture, dynamic behavior, timbral quality. These are assembled into a structured prompt formatted specifically for AI music generation tools. The user's intuitive response to real music becomes precise technical language.

The Match / Contrast Logic

The reference library is split into two categories by design. Match references reinforce the scene's emotional register — tension meets tension, melancholy meets melancholy. Contrast references work against it deliberately. Some of the most effective game audio decisions create meaning through opposition: a cheerful melody over horror, a lullaby over violence. Giving users both directions forces the creative decision to be conscious rather than defaulted.

Match — Emotional Reinforcement

Music that amplifies what the scene already feels. Tension meets tension. Melancholy meets melancholy. The emotional logic is transparent — the scene and the music agree about what is happening.

Contrast — Deliberate Opposition

Music that works against the scene's surface register. A cheerful waltz over a horror sequence. A lullaby over isolation. The gap between image and sound generates unease or irony that neither element produces alone.

Design Principle

This tool treats the human's inability to name musical qualities not as a deficit to be educated away but as a fixed condition to be designed around. The workflow meets creators where they are. The output — a structured Gemini Music prompt — contains only musical parameters, never narrative descriptions. What goes in as feeling comes out as tempo, key, instrumentation, and texture. The translation happens inside the tool, not inside the user's head.

What goes in

"A tense chase through a neon-lit alley"

"Melancholy, like saying goodbye"

"Epic but hollow — a victory that feels wrong"

What comes out

Fast-paced driving strings at 140 BPM, staccato brass stabs

Sparse piano, minor key, 55–65 BPM, sustained legato phrases

Full orchestral swell, major key, sudden dynamic drop, cold reverb

Origin

This tool was built in response to a concrete task: designing game audio for TokenVerse. Facing the challenge of generating music that matched specific narrative moments, I identified the exact breakdown point between human creative intent and AI music generation capability — and built a system to bridge it. The tool is now a published Claude Skill, available as a reusable workflow for any project requiring AI-assisted audio composition.
